# 🚀 Alien Invasion Game

An arcade-style space shooter game developed using Python and Pygame.

## 🎮 About The Game
In this game, the player controls a spaceship and fights against waves of invading aliens.  
The goal is to survive as long as possible, destroy enemies, and achieve the highest score.

## ✨ Features
- Smooth player movement
- Bullet shooting system
- Multiple alien enemies
- Score tracking system
- Increasing game difficulty
- Sound effects and animations
- Game over and restart system

## 🛠 Technologies Used
- Python
- Pygame

## 📷 Gameplay
Fast-paced action gameplay inspired by classic arcade space games.
